About Us

Voice Communication Control System (VCCS)

This VCCS system is for use by VATSIM Air Traffic Controllers with VATSIM UK. You will require an active VATSIM account to use this system and be logged onto the network as a controller using a client such as Euroscope. Not for real world use.

Privacy Policy

No personal data is stored by VCCS. Authentication takes place using the VATSIM Single Sign-On system. This system then passes transient data which is stored only in your browser using session cookies.

VCCS will match your VATSIM CID with publically available data from https://data.vatsim.net/ to verify the position you are controlling.

We will NOT share, sell or otherwise allow access to your data with any third-party subject to the statement above.
